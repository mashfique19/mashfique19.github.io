## Overview

This project involved the design and simulation of a modern household network using Cisco Packet Tracer. The objective was to create a reliable and scalable network architecture capable of supporting multiple wired and wireless devices while ensuring efficient communication and internet connectivity.

The network was designed to accommodate typical home applications including web browsing, video streaming, file transfers, online gaming, email communication, and distance learning.

---

## Objectives

* Design a complete home network topology
* Configure wired and wireless connectivity
* Support multiple device types
* Ensure reliable communication between devices
* Simulate real-world network behavior
* Apply network design principles using Cisco Packet Tracer

---

## Network Components

The network included the following devices:

* 2 Wireless Routers (WRT300N)
* 2 Desktop PCs
* 2 Laptops
* 3 Smartphones
* 2 Smart TVs
* Network Switch
* DSL Modem
* Cloud Connection
* Server

---

## Network Architecture

The network was designed around two wireless routers that served as the primary connectivity hubs.

### Wired Devices

* Desktop PCs connected via Ethernet
* Network switch for device interconnection
* DSL modem for internet access

### Wireless Devices

* Laptops
* Smartphones
* Wireless client connectivity

### Multimedia Devices

* Smart TVs configured for video streaming services

The design ensured efficient traffic distribution and stable connectivity for all connected devices.

---

## Bandwidth Planning

| Device Type | Connection    | Main Applications                                   | Estimated Bandwidth |
| ----------- | ------------- | --------------------------------------------------- | ------------------- |
| Smartphone  | Wireless      | Email, File Transfer, Web Browsing                  | 20 Mbps             |
| Laptop      | Wireless      | Web Browsing, Video Conferencing, Distance Learning | 40 Mbps             |
| PC          | Ethernet      | Email, Web Browsing, Online Gaming                  | 40 Mbps             |
| Smart TV    | Coaxial/Wired | Video Streaming                                     | 20 Mbps             |

Bandwidth allocation was planned to support simultaneous usage of multiple applications while maintaining network performance.
